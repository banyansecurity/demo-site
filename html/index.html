FAQ

How does Zero Trust Remote Access work?
Banyan Security’s Zero Trust Remote Access platform enables employees and third parties to securely access on-premise, hybrid, and multi-cloud applications and servers without the need to use traditional VPNs. Read more about Banyan’s Zero Trust platform.
How do I sign up / set up for the Banyan Security Test Drive?
Please contact your Banyan Account Representative to get your MedSoft credentials. Then, head on over to the Setup Guide to get set up.
What access privileges and roles do I have in the Banyan Security Test Drive?

You are assigned admin access to the Banyan Command Center for the Test Drive organization - see how to publish a catalog of corporate services for end users and define Zero Trust policies. Also check out our real-time events and reporting views.

You will also install the Banyan App as an end user, so you can get a feel for the end user registration experience and one-click access flows.

Finally, you have limited access to all the MedSoft corporate services.

Why do I need to install the Banyan App for the Test Drive?

One of the cornerstones of a Zero Trust security model is the ability to establish Device Trust. In Google’s BeyondCorp security model, for example, only devices that are procured and actively managed by the enterprise can access corporate applications.

While Banyan's Zero Trust Remote Access solution supports both managed and unmanaged (BYOD, third-parties, etc) scenarios, our fictional company, MedSoft Digital, enforces a security model where corporate applications should only be accessed by Registered Devices. Installing the Banyan App allows you to register your device and access various MedSoft corporate services. You can also get a good feel for the user experience you will deliver to your end users.

Review our product docs to learn more about device registration and considerations when deploying the Banyan Apps across your workforce’s devices.

Will my data and information be private?

Yes! We obfuscate device-specific data (such as serial number or UDID) and randomly generate account identifiers.

Read more about data privacy related to the Banyan App and our Platform.

How do I completely remove the Banyan App from my device when I'm done with Banyan Test Drive?

Uninstalling the Banyan Desktop App leaves behind a small data folder containing application state. This allows you to upgrade or reinstall the app without having to go through the device registration process again.

You can easily remove this small data folder by following our OS-specific instructions.
